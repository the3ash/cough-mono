# Cough Mono

A crisp and sharp monospaced typeface.

## Character Variants

- `cv01`: Alternative `a`
- `cv02`: Alternative `f`
- `cv03`: Alternative `f` 2
- `cv04`: Alternative `i`
- `cv05`: Alternative `r`
- `cv06`: Alternative `y`
- `cv07`: Alternative `Q`
- `cv08`: Alternative `1`
- `cv09`: Dotted `0`
- `cv10`: Plain `0`
- `cv11`: Alternative `@`
- `cv12`: Alternative `*`
- `cv13`: Alternative `:`
