# Cough Mono

A crisp and sharp monospaced typeface.

## Character Variants

- `cv01`: Alternative `a`
- `cv02`: Alternative `i`
- `cv03`: Alternative `r`
- `cv04`: Alternative `y`
- `cv05`: Alternative `Q`
- `cv06`: Alternative `1`
- `cv07`: Dotted zero
- `cv08`: Plain zero
- `cv09`: Alternative colon
