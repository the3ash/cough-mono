# Cough Mono

A crisp and sharp monospaced typeface.

## Character Variants

- `cv01`: Alternative `a`
- `cv02`: Alternative `f`
- `cv03`: Alternative `i`
- `cv04`: Alternative `r`
- `cv05`: Alternative `y`
- `cv06`: Alternative `Q`
- `cv07`: Alternative `1`
- `cv08`: Dotted `0`
- `cv09`: Plain `0`
- `cv10`: Alternative `*`
- `cv11`: Alternative `:`
