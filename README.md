# Cough Mono

A crisp and sharp monospaced typeface.

## Character Variants

- `cv01`: Alt `a`
- `cv02`: Alt `f` 1
- `cv03`: Alt `f` 2
- `cv04`: Alt `i`
- `cv05`: Alt `r`
- `cv06`: Alt `y`
- `cv07`: Alt `Q`
- `cv08`: Alt `1`
- `cv09`: Dotted `0`
- `cv10`: Plain `0`
- `cv11`: Alt `@`
- `cv12`: Alt `*`
- `cv13`: Alt `:`
