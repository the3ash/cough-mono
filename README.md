# Cough Mono

A crisp and sharp monospaced typeface.

## Use via npm

```sh
npm i @the3ash/cough-mono
```

```js
import "@the3ash/cough-mono";
```

```css
body {
  font-family: "Cough Mono", monospace;
}
```

## Character Variants

- `cv01`: Single-story `a`
- `cv02`: Descending `f`
- `cv03`: Footed `f`
- `cv04`: Footed `i`
- `cv05`: Rounded-shoulder `r`
- `cv06`: Straight-tail `y`
- `cv07`: Angled-tail `Q`
- `cv08`: Angled `1`
- `cv09`: Dotted `0`
- `cv10`: Plain `0`
- `cv11`: Centered `:`
- `cv12`: Full-a `@`
- `cv13`: Six-arm `*`

## Stylistic Sets

- `ss01`: Classic forms - Enables `cv03` through `cv08` together.
